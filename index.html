Privileges in the OpenLab CDS (ver 2.3 - 3.0) Create by R2 KA 1Team

The following tables list the privileges that are associated with OpenLab CDS users though the Control Panel for Project privileges, Instrument privileges, and Administrative privileges.

다음 표는 Control Panel을 통해 OpenLab CDS 사용자에게 적용되는 Project 권한, Instrument 권한, Administrative 권한과 관련된 Privilege를 설명합니다.

Administration > Role 항목에서 권한(Privileges)을 설정할 수 있으며, 설정된 권한은 사용자(User) 또는 그룹(Group) 단위로 적용합니다.

PROJECT PRIVILEGES

Table 4: Project Management

Name Description
Manage project or project group User can create or edit project properties and can move the project but cannot view or edit the project access settings. 사용자는 프로젝트 속성을 만들거나 편집할 수 있으며 프로젝트를 이동할 수는 있지만 프로젝트 엑세스 설정에 대해 확인하거나 편집할 수 없다.
Manage project or project group access User can view and edit the project access settings. This is for "edit privileges" when user right click the location selected. 사용자는 프로젝트 액세스 설정을 보고 편집할 수 있다.
View project or project group User can see a project and project details but cannot edit. Note: This privilege is required for all users. 사용자는 프로젝트 및 프로젝트 세부 정보를 볼 수 있지만 편집할 수는 없다. *고정된 권한으로 권한 해제 불가
Access content using web client User can view the data via the Content Management web interface. 사용자는 Content Management Web 화면에서 데이터를 볼 수 있다.
Edit content of project User can create new versions of documents (e.g. data, methods, or templates). 사용자는 새로운 버전의 문서를 작성할 수 있다. CDS Project 에 Data, Method, Sequence등을 생성할수 있으며, 사용자는 data 생성을 위해 해당 권한을 필요로한다. *해당 권한을 부여받지 않은 경우 Method, Sequence 저장 아이템이 비활성화되어지며, 분석또한 불가하다.

Table 5: E-Signature

Name Description
E-Signature Sign Data Files User can sign data files 사용자는 데이터 파일에 서명할수 있다. *해당권한이 부여되지 않은 경우 전자 서명 항목은 비활성화가 아닌 아이템이 보여지지 않는다.
Revoke E-Signature User can revoke the e-signature. 사용자는 전자 서명을 취소할 수 있다. *전자서명 이후 해당 아이템이 활성화 되어 지며, 해당 권한을 부여 받지 않은 경우 비활성화로 표시되어 진다.

Table 6: Sample Prep

Name Description
Create and modify sample prep View, edit, and save an autosampler sample prep file Autosampler sample prep 파일의 보기, 수정,저장 *Acquisition 에서 windows 항목 중 sample prep에 접근시 해당 권한 부여 시 sample prep method 를 생성, 수정 그리고 저장 할 수 있다.

Table 7: Acquisition Method

Name Description
Create and modify acquisition method Create, edit and save an acquisition method file (*.amx) AMX(acquisition method) 의 생성, 수정, 저장 *해당 권한이 부여되지 않은 경우 저장, 새로운 이름으로 저장이 불가 하다.

Table 8: Processing Method

Name Description
Create processing method Create a new processing method (*.pmx), or save a method under a new name. 새로운 PMX(Processing method) 를 생성하거나 새로운 이름으로 저장할 수 있다.
Save Master Method Save changes to a processing method in the Methods folder. CDS Project Method 폴더에 생성된 Master Method 에 대한 변경 내용을 저장할 수 있다.
Save result set method Save changes to a processing method in the result set folder. CDS Project Result set 폴더에 생성된 Result Set Method 에 대한 변경 내용을 저장할 수 있다.
Edit integration parameters View and edit the parameters in the Integration Events section of a method. method항목중 integration event 에대해 확인하고, 편집 할 수 있다. *Standard, Advanced 파라미터 모두 비활성화 상태로 표시.
Edit identification parameters View and edit the parameters in the Compounds > Identification section of a method. Method 항목 중 compounds 항목의 identification 의 파라미터를 확인하고, 편집 할 수 있다.
Edit chromatogram extraction parameters View and edit the parameters in the Extraction > Chromatogram section of a method. Method 항목 중 extraction 항목의 chromatogram section 파라미터를 확인하고, 편집 할 수 있다. *Extraction 을 지원하는 method 에 한함( ex. 3D method)
Edit spectrum extraction parameters View and edit the parameters in the Extraction > Spectrum section of a method. Method 항목 중 extraction 항목의 spectrum section 파라미터를 확인하고, 편집 할 수 있다. *Extraction 을 지원하는 method에 한함(ex. 3D method)
Edit MS library search parameters View and edit the parameters in the MS Library Search > Properties section of a method. Method 항목 중 MS Library Search 항목의 Properties 파라미터를 확인하고 편집할 수 있다. *MS library 를 지원하는 경우에 해당 아이템 확인 가능
Edit calibration parameters View and edit the parameters in the Compounds > Calibration section of method. Method 항목 중 Compounds 항목의 Calibration 파라미터를 확인하고 편집할 수 있다. *Calibration 이 가능한 method 에 한함.
Edit spectra parameters View and edit the parameters in the Compounds > Spectra section of method. Method 항목 중 Compounds 항목의 Spectra 파라미터를 확인하고 편집할 수 있다.
Edit system suitability parameters View and edit the parameters in the Compounds > System Suitability section of method. Method 항목 중 compounds 항목의 system suitability 파라미터를 확인하고 편집할 수 있다. *System suitability 를 지원하는 method 에 한함.
Edit custom calculation parameters View and edit the parameters in the Tools > Custom Calculation section of a method. Method 항목 중 tools 항목의 custom calculation 파라미터를 확인하고 편집할 수 있다.
Edit signal parameters View and edit the parameters in the General > Signals section of a method. Method 의 general 항목의 signals 파라미터를 확인하고 편집할 수 있다.
Edit sample purity parameters View and edit the parameters in the MS Sample Purity section of a method. Method 항목의 MS sample purity 파라미터를 확인하고 편집할 수 있다. LC/MS Sample Purity DA method 에 해당되며, MS sample Purity Properties 항목이 비활성화로 표시
Edit reporting parameters View and edit the parameters in the Reports > Injection Report section of a method. Method 항목의 report 항목의 injection report 파라미터를 확인하고 편집할 수 있다. *해당 권한을 부여받지 않은 경우 injection report 아이템이 비활성화로 표시된다.
Edit general parameters View and edit the parameters in the General > Properties section of a method. Method 항목의 general 항목의 properties 파라미터를 확인하고 편집할 수 있다. *info 항목의 comment 입력, global 항목에서 integrator 에 대한 수정
Load older master method With Content Management, load an older version of a master method. 이전 버전의 master method 를 불러올수 있다. *해당 권한을 부여 받지 않은 경우 method open 진행 시 하단에 이전 ver 에 대한 선택 option 이 비활성화 상태이다.

Name Description
Unlock/lock report template items Lock and unlock report template items (tables, chromatograms, groups of items, ...) to control who is allowed to modify those. Report items 항목에서(표, 크로마토그램, 그룹 등) 을 잠금, 잠금해제하여 수정할 수 있다. *해당 권한이 부여된 경우 아이템 선택 시 잠금/ 잠금해제 항목이 활성화 되어진다.
Validate report template Confirm usage of report templates that have been modified outside OpenLab CDS. 외부에서 수정된 report templates의 사용 여부를 확인 할 수 있다. Accept template 를 선택하지 않은 경우 template 을 불러오기 할 때마다 메시지가 표시되어지며, 다른 이름으로 저장한 경우에도 Accept 전까지 메시지가 발생된다. Accept 되지 않은 report 로 결과처리 진행시 경고기호 워터마크가 함께 표시되어진다.
Create report template Create and edit report templates in the Reporting view. Report 를 생성하고 편집 할 수 있으며, 리포트 미리보기를 제공한다.(해당 권한 부여시 Default Report Temple을 추가 할 수 있다. 권한 미 부여시 defaults Temple import item 표시 되지 않는다)

Table 10: Sequence Template

Edit users own running sequences Edit your own running sequences (status Acquiring in the Run Queue). Acquiring 상태에 있는 sequence 를 수정할 수 있다. *해당 권한이 부여되지 않은 경우 edit 항목은 활성화 되어 있으며, sequence를 수정할 수 있지만, update 가 불가하다.

Table 12: Audit Trail

Name Description
Change method audit trail settings Edit and save method audit trail settings (project properties in the Control Panel). Control panel > project 항목에서 method audit trail 설정을 편집하고 저장 할 수있다. *Instrument method 항목에 Activated 항목이 비활성화 된다.(Project 항목에서 audit trail 설정이 진행되지 않은 경우)
Review audit trail Confirm that you reviewed a changed audit trail. 변경된 audit trail 에 대한 review 를 진행 할 수 있다. *해당 권한이 부여되어 있지 않는 경우 Review 아이템이 보이지 않는 상태로 표시된다.
Add manual audit trail entry Add a manual entry to document your own actions in the audit trail. 수정된 항목에 대해 사용자가 수동으로 audit trail 을 남길수 있다. *method, sequence, project audit trail 에 해당

Table 13: Control

Name Description
Abort any running sample Abort any running sequence or single run. 다른 사용자가 실행 중인 sequence또는 single 분석에 대해 about 할수 있다. *다른 사용자에 의해 실행중인 sequence, single run 에 해당
Manual control (in run) Access manual control functions while the instrument is running. 분석 중인 상태에서 manual control 항목에 접근할수있다.
Manual control (only when instrument idle) Access manual control functions while the instrument is idle. Idle 상태에서 manual control 항목에 접근할 수 있다. *권한을 부여 받은 사용자는 Instrument Status 에서 Method 항목에 접근 할 수 있으며, Turn devices on 아이템이 활성화 되어 진다.
MS autotune and manual tuning Access all MS tune and maintenance functionality, including manual tune, autotune, and check tune. 수동 또는 자동 tune 및 체크 tune 을 포함한 모든 MS tune 및 유지관리 기능에 접근 할 수 있다.
MS autotune Perform MS autotune and check tune. Auto tune 을 지원하고 tune 상태를 확인 할 수 있다.
Delete any pending runs Remove pending runs from the Run Queue. 다른사용자가 분석을 위해 실행시킨 Run Queue 항목의 pending 상태에 있는 항목에 대해 제거 할 수 있다. 해당 기능 삭제되고, CDS 2.7 에서 Cancel my pending run, Cancel any pending run 으로 대체
Reorder pending runs Change order of pending runs in the run queue. Run Queue 항목의 list에 대해 실행 순서를 변경 할 수 있다.

Print results reports Create reports for your methods or results. Method 또는 results 에 대한 결과 report 를 생성할 수 있다. *해당권한이 부여되지 않는 경우 reports 항목의 print 아이템이 보여지지 않는다.
Launch Custom Calculation Editor Start the Custom Calculation Editor from Data Analysis. Custom Calculation Editor 를 실행 할 수 있다. *해당 권한이 부여되지 않는 경우 tools 항목의 Launch CC Editor 아이템이 보여지지 않는다.

INSTRUMENT PRIVILEGES

Table 20: Instrument Management

Name Description
View instrument or location User can view and access a location in the tree, but not edit access security, can view properties. 사용자는 트리에서 위치를 보고 액세스할 수 있지만 액세스 보안을 편집할 수는 없으며 속성을 볼 수 있다. *고정된 권한으로 권한 해제 불가
Manage Instrument or location User can create and move locations and edit properties (name, description, etc). 사용자는 위치를 생성하고 이동할 수 있으며 속성(이름, 설명 등)을 편집할 수 있다.
